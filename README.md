# mc558_2s2025_ra253769
MC558 Laboratório

Como executar os arquivos cpp

Para rodar os testes coloque dentro da pasta raiz a pasta input/ com todos as entradas : arq01.in...

Para executar os testes se as saidas estao corretas utilize

make allTests

Para executar os testes com tempo de execucao cronometrado utilize

make allTimers

Para executar o algoritmo em Python com Programação Linear, tenha a licença gurobi configurada

Inicie uma nova venv, e instale pip install -r requirements.txt

com a venv ativa roda os testes da seguinte forma

Para os testes se está correto !

make plShortestPath

Para os testes com tempo de execução cronômetrada

make timerPlShortestPath
